# 폰트

이 폴더에 **Pretendard** 폰트 파일(`Pretendard-*.ttf` 또는 `.otf`)을 넣으면
앱이 자동으로 인식해 사용한다. 없으면 Windows 기본 한글 폰트인
**Malgun Gothic** 으로 폴백한다.

- 다운로드: https://github.com/orioncactus/pretendard/releases
  (`Pretendard-Regular.ttf`, `Pretendard-Bold.ttf` 등을 여기에 복사)
- 라이선스: SIL Open Font License 1.1

자간(-0.3px)은 폰트와 무관하게 코드에서 적용된다(`augtool/ui/style.py`).
